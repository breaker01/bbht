## Installing
Make sure you manually install Go before running this tool. You can view the Go docs [here](https://golang.org/doc/install).

Run the following commands:
```
$ git clone https://github.com/JR0ch17/bbht
$ cd bbht/
$ chmod +x install.sh
$ ./install.sh
```
